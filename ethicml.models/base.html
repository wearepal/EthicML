call_script(cmd_args, env=None, cwd=None)#

Call a (Python) script as a separate process.

An exception is thrown if the called script failed.

Parameters:

  • cmd_args (list[str]) – List of strings that are passed as commandline arguments to the executable.

  • env (Optional[dict[str, str]]) – Environment variables specified as a dictionary; e.g. {"PATH": "/usr/bin"}.

  • cwd (Optional[Path]) – If not None, change working directory to the given path before running command.

Return type:

None

property executable: str#

Path to a (Python) executable.

By default, the Python executable that called this script is used.
